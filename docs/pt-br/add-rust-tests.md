# Adicionando testes em Rust

### 11. Finalizando o projeto com testes em Rust

Agora que voce tem todos os endpoints e testes passando, voce pode usar testes em Rust para validar a corretude. Os testes de shell foram suficientes para validar Python e Rust usando a API HTTP. Mas agora voce pode usar testes em Rust para validar a corretude do projeto Rust usando seus proprios testes.

- Pergunte ao Copilot onde voce pode adicionar testes para o projeto Rust. Testes podem ficar no mesmo arquivo `main.rs` ou em um arquivo separado.
- Pergunte como executar os testes para validacao
- Adicione apenas um teste por vez e valide. Esse e o mesmo processo de antes e vai ajudar voce a focar em uma coisa de cada vez.

??? question "Tip"
    Prompt *(Agent Mode)*

    ```text
    Now that we have all endpoints in Rust I want native Rust unit tests. Add them
    to main.rs so that I can verify correctness without having to use the BASH file
    ```

Voce deve ter um aplicativo Rust totalmente funcional e bem testado agora.
Se voce chegou ate aqui, parabens! Voce pode tentar realizar essas tarefas antes que o tempo acabe ou tentar depois no seu proprio tempo.
