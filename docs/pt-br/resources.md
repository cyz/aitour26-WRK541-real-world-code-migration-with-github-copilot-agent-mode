## Pre-requisitos para ambiente local

Se voce quiser executar o workshop localmente em vez de usar GitHub Codespaces, garanta que o seguinte esteja instalado na sua maquina (assumimos que estao prontos para voce iniciar os exercicios):

- Uma conta GitHub com GitHub Copilot habilitado no VS Code
- Git para clonar o repositorio
- Visual Studio Code
- Python 3.12 e `pip` (para instalar `src/python-app/requirements.txt`)
- .NET 10 SDK (para a Minimal API em C#)
- Opcional: extensao REST Client para testar endpoints no VS Code

Para passos detalhados de instalacao e verificacao, veja o checklist completo de pre-requisitos no repositorio: [Local Development Prerequisites](../../PREREQUISITES.md).
