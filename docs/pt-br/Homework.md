# Tarefa de casa

Parabens por concluir o workshop! Para consolidar seu aprendizado e explorar conceitos adicionais, aqui estao alguns exercicios e projetos estendidos.

### Desafio para casa: Implantar a aplicacao no Azure

Se voce concluiu todas as etapas anteriores, incluindo a containerizacao, tente
implantar sua aplicacao no Azure. Para isso, voce precisara de uma conta Azure
e da extensao GitHub Copilot for Azure
extension](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azure-github-copilot)
que vai ajudar voce a fazer perguntas e obter orientacao para o deployment.

![Extension Page for GitHub Copilot for Azure](./media/gh-copilot-for-azure.png)

1. Use `@azure` depois de fazer login na sua conta Azure para perguntar como implantar o aplicativo C# containerizado
1. Use um dos servicos sugeridos para implantacao, por exemplo, Azure Container Apps ou Azure App Service, para implantar o container
1. Verifique se o container esta implantado e funcionando

Ao final, garanta que voce limpe e remova todos os recursos para evitar cobrancas desnecessarias na sua conta.
